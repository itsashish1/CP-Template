# CP-Template

> Competitive Programming Templates for Codeforces, CodeChef, LeetCode, and other online judges

## Description

This repository contains pre-written algorithms, data structures, and utility functions that serve as a ready-to-use template for competitive programming contests.

## Contents

- **Fast I/O Template** - Optimized input/output for C++
- **Data Structures** - Segment Tree, Fenwick Tree, DSU, etc.
- **Algorithms** - Sorting, Searching, Graph Algorithms, DP templates
- **Number Theory** - Prime sieve, GCD, Modular arithmetic
- **Geometry** - Basic geometric utilities

## Usage

Copy the relevant template code and modify as needed for your contest solution.

## Languages

- C++ (Primary)
